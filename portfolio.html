Shrink'n'Grow (Spring 2022)

This project was developed as part of my CSC 4263 course (Video Game Design). It marked my first experience working with a game engine and was completed within a short two-week development timeline.

The project was developed in Unity, utilizing Microsoft Visual Studio as the IDE and C# as the programming language, with Cinemachine as the only imported asset. It was an individual project designed to introduce students to the Unity game engine, and to allow students to show off their stengths before forming a group.

Shrink'n'Grow is a 3D third-person puzzle-platformer that revolves around the main mechanic of shrinking and growing the player character. Inspired by my love for games that masterfully convey a sense of scale, I designed simple puzzles that challenge players to creatively utilize their size-changing abilities to progress. A key focus during development was refining the camera controls to ensure smooth and intuitive player navigation, enhancing overall gameplay experience.

If I were to return to this project, I would focus on refining the size-changing mechanic to ensure the camera dynamically adjusts while maintaining consistent relativity to the player's size. Additionally, I would dive deeper into level design, creating expansive, larger-than-life environments that enhance the sense of scale. Echos (Spring 2022)

Website

Echos was developed by a small team of four developers and one artist as part of our CSC 4263 (Video Game Design) course. The project had a three-month timeline to propose, design, and develop the game.

Created in Unity, Echos utilized Plastic SCM for version control and collaborative project management. As one of the developers, my primary focus was on player movement, camera controls, and refining prototype mechanics. Drawing from my experience as a playtester, I also contributed to bug testing and fixing issues throughout the development process.

Echos is a third-person 3D puzzle game where players control an adventurer on a quest for treasure within a decaying, abandoned castle. The adventurer can possess the bodies of long-deceased heroes from the past, altering the course of a tragic conflict. Each hero brings unique abilities that are vital for solving puzzles, enabling the adventurer to overcome obstacles and advance through the castle in the present.

Logan's Sandbox (Fall 2024)

Repository

I developed this project to explore and implement some of my favorite game mechanics, with the goal of making them reusable for future projects.

This project is made in Unity 6.

The first feature I developed for this project was creating seamless portals for an FPS game, inspired by Valve's Portal. I used a reference video to learn the basics of portal implementation and made modifications to allow portals to function independently, without needing to be directly linked. This project taught me valuable concepts, including matrix formulas for calculating relative positions, the fundamentals of UV texture mapping, and an introduction to shader programming.

For future development, I plan to explore additional game mechanics inspired by some of my favorites, such as wall running, seamless vehicle entry and exit, melee combat, and designing an RPG-style leveling system, among others.

Cave Has Eyes (Winter 2024)

This is a collaborative project with my partner, aimed at learning Unreal Engine, exploring common JRPG mechanics, and experimenting with a mysterious and horrific setting.

This project is being developed in Unreal Engine 5, with GitHub used for version control and collaboration.

This project is currently in the early stages of development, with the setting and theme being the most developed aspects so far. The game will feature Lovecraftian horror elements and an overarching sense of mystery. Its core gameplay will revolve around turn-based strategy, with a leveling system based on completing events rather than grinding experience from defeating enemies. Outside of combat, players will engage in 3D exploration, with a strong emphasis on items and quests to drive progression.
